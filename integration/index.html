Exercice intégration

Entraînement checkpoint 1

Des instructions sont données pour "embellir" chaque section.

Ajoute tes CSS dans le fichier style.css fourni.

1. En-tête

pour commencer, embellis l'en-tête ci-dessus en :

  1. Ajoutant de l'espacement au-dessus, en-dessous
  2. Centrant son texte
  3. Ajoutant une image de fond (suggestions... mais prends un fond sombre)
  4. Mettre le texte en couleur claire pour garder du contraste

2. Mise en page générale

Fais en sorte que les sections (pas leur contenu !) soient centrées dans la page, et ne dépassent pas une certaine largeur (48em).

Supprime les marges mises par défaut par le navigateur sur toute la page.

3. Barre de navigation

Il n'y a pas de navbar !

Crées-en une de toutes pièces !

  1. Place-la juste sous le header.
  2. Utilise une balise sémantique appropriée pour englober ce qu'elle va contenir.
  3. Dans cette balise, crée un <ul>, avec à l'intérieur des <li>, avec à l'intérieur des <a>.
  4. Cible les ancres présentes dans cette page.
  5. Supprime les "bulles" devant les items
  6. Aligne les items
  7. Fais en sorte que la couleur des liens change quand on les survole.

4. Responsive

Dans cette section se trouvent des "cards".

  1. Fais en sorte qu'elles s'affichent :
    1. Sur une colonne pour un affichage de largeur inférieure à 480px.
    2. Sur deux colonnes pour un affichage de largeur comprise entre 480px et 991px.
    3. Sur trois colonnes pour un affichage de largeur supérieure ou égale à 992px.
  2. Tu dois utiliser flexbox et des media queries !
  3. Calcule les largeurs de façon à laisser un peu d'espace entre les cards horizontalement...
  4. Laisse aussi un peu d'espacement vertical...
  5. Ajoute une border ou une ombre aux "cards"
  6. Fais en sorte que leur contenu ne soit pas collé au bord !
